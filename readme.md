# 🚀 FreeQwenApi2 — Инструкция для новичков (на пальцах)

Эта библиотека позволяет бесплатно использовать функции сайта Qwen (ИИ от Alibaba) прямо через твой Python-код. Ты можешь получать список моделей, общаться текстом, генерировать картинки и видео.

---

## 🛠️ Как это установить и настроить

1. **Папки проекта** должны выглядеть строго вот так:
   ```text
   FreeQwenApi2/
   ├── cookies.txt          <-- Сюда кидаешь куки из браузера (в формате Netscape)
   ├── main.py             <-- Твой главный рабочий скрипт
   └── qwen_api/           <-- Папка с библиотекой
       ├── __init__.py
       ├── qwen_models.py
       ├── qwen_text.py
       ├── qwen_image.py
       └── qwen_video.py

2. **Где взять куки (`cookies.txt`)?**
Зайди в свой аккаунт на `chat.qwen.ai`, открой любое расширение в браузере для экспорта куков (например, *Get cookies.txt LOCALLY* или *Cookie-Editor*) и экспортируй куки в формате **Netscape**. Переименуй файл в `cookies.txt` и положи в корень папки.

# 📖 Шпаргалка по функциям: куда бить и что прилетит

## 1. Сбор доступных моделей (Без авторизации)

Нужно, чтобы узнать, какие нейросети сейчас вообще работают на сервере.

* **Куда запрашивать:** `get_available_models()`
* **Что нужно передать:** Ничего не нужно.
* **В каком виде вернется ответ:** Прилетит огромный сырой JSON (словарь), в котором внутри ключа `data` лежит всё — названия моделей, их лимиты и умения.

**Как вызывать в коде:**

```python
from qwen_api import get_available_models

# Просто вызываем функцию
все_модели = get_available_models()
print(все_модели)

```

## 2. Текстовый чат (Контекст и память)

Нужно для обычного текстового общения. Бот автоматически помнит, о чем вы говорили сообщение назад.

### Шаг А: Создать чат

Прежде чем писать ИИ, нужно создать для него «комнату» (сессию).

* **Куда запрашивать:** `ai.create_chat()`
* **Что нужно передать:** Ничего.
* **В каком виде вернется ответ:** Чистая строка с уникальным ID чата (например: `"791cb19f-54b8-4c4a-a944-42f1477bd129"`). Если вернулся `None` — твои куки сдохли.

### Шаг Б: Отправить сообщение

* **Куда запрашивать:** `ai.ask(chat_id, text_message)`
* **Что нужно передать:**
* `chat_id` — тот самый ID комнаты, который мы получили на Шаге А.
* `text_message` — твой текст (например: *"Привет, как дела?"*).


* **В каком виде вернется ответ:** Ответ летит **потоком (стримом)**. Это значит, ИИ возвращает буквы по очереди, прямо как на сайте. Код будет ловить их по кусочкам.

**Как вызывать в коде:**

```python
from qwen_api import QwenChatAPI

# 1. Включаем текстовый движок и даем ему куки
ai = QwenChatAPI(cookies_file="cookies.txt")

# 2. Создаем комнату для чата
id_чата = ai.create_chat()

if id_чата:
    # 3. Отправляем вопрос и сразу печатаем кусочки ответа на экран
    for кусок_текста in ai.ask(id_чата, "Расскажи короткую шутку"):
        print(kuсок_текста, end="", flush=True)

# 4. Выключаем за собой ИИ
ai.close()

```

## 3. Генерация картинок (Text-to-Image)

Нужно, если ты хочешь, чтобы нейросеть что-то нарисовала.

* **Куда запрашивать:** `img_ai.generate_image(chat_id, prompt, aspect_ratio)`
* **Что нужно передать:**
* `chat_id` — ID чата (лучше использовать свежесозданный чат).
* `prompt` — описание картинки (*"рыжий кот в космосе"*).
* `aspect_ratio` — формат. На выбор можно писать: `"1:1"` (квадрат), `"16:9"` (горизонтальная), `"4:3"`.


* **В каком виде вернется ответ:** Чистая **прямая ссылка** на картинку в формате `.png` с серверов Qwen (например: `https://cdn.qwenlm.ai/output/...png`). Ты можешь сразу вставить её в браузер или скачать.

**Как вызывать в коде:**

```python
from qwen_api import QwenChatAPI, QwenImageAPI

chat_ai = QwenChatAPI()
img_ai = QwenImageAPI()

# Картинкам тоже нужен ID чата, создаем его
id_чата = chat_ai.create_chat()

# Запускаем генерацию
ссылка_на_фото = img_ai.generate_image(chat_id=id_чата, prompt="Капибара в короне", aspect_ratio="1:1")

print("Ссылка на твою картинку:", ссылка_на_фото)

chat_ai.close()
img_ai.close()

```

## 4. Генерация видеороликов (Text-to-Video)

Самая тяжелая функция. Так как видео делается долго, скрипт отправит запрос, а потом сам будет каждые 5 секунд спрашивать у сервера: *"Ну что, готово? А сейчас? А сейчас?"*, пока сервер не отдаст видео.

* **Куда запрашивать:** `video_ai.generate_video(chat_id, prompt)`
* **Что нужно передать:**
* `chat_id` — ID чата.
* `prompt` — что должно происходить на видео (*"киберпанк машина едет в тумане"*).


* **В каком виде вернется ответ:** Чистая **прямая ссылка** на готовый видеофайл в формате `.mp4` (например: `https://cdn.qwenlm.ai/output/...mp4`). Генерация обычно занимает от 1 до 3 минут, просто жди завершения работы скрипта.

**Как вызывать в коде:**

```python
from qwen_api import QwenChatAPI, QwenVideoAPI

chat_ai = QwenChatAPI()
video_ai = QwenVideoAPI()

id_чата = chat_ai.create_chat()

print("Пошел рендеринг видео, это займет около двух минут...")
ссылка_на_видео = video_ai.generate_video(chat_id=id_чата, prompt="Взрыв сверхновой звезды, космос, реализм")

print("Видео готово! Ссылка на скачивание:", ссылка_на_видео)

chat_ai.close()
video_ai.close()

```

## ⚠️ Коротко о главном (Почему что-то может пойти не так):

1. **Если всё сломалось и выдает ошибку:** Проверь файл `cookies.txt`. Скорее всего, ты разлогинился на сайте или скопировал куки не в том формате.
2. **Если вместо текста вылезают пустые скобки `[]`:** Сервер временно перегружен или заблокировал твой IP. Попробуй включить/выключить прокси или VPN.
